# Docker container for GitHub Copilot CLI

Launch a Copilot CLI docker image on an app directory

### To launch Copilot CLI, Just run:
```
    sh copilot_cli.sh <path_to_project_directory>
```

### Once the shell prompt appears, just start copilot:
```
    ubuntu@aa11bbccfff:/$ copilot
```

[This](https://docs.github.com/en/copilot/how-tos/copilot-cli/cli-best-practices) might be a useful place to start if you're new to Copilot CLI
